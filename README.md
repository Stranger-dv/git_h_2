# Git

# H-2 (Enviar un directorio vacio al repositorio remoto)

## Comandos utilizados
```
git init
git remote add origin (pegar_la_ruta_del_repositorio_local)
git remote -v
mkdir usuarios
ls -l
git status
cd usuarios
touch .gitkeep
cd ..
ls -l
git status
git add .
git commit -m "feat: add directory usuarios"
git log --oneline
git branch -M main
git push -u origin main
```
